| type          | \*\*\*StartTimeNanos        | データの取り扱い                                                         |
| ------------- | --------------------------- | ------------------------------------------------------------------------ |
| Weight        | データ上1日に複数回存在する | 同じ日の新しい時間のデータがあっても更新しない                           |
| BloodPressure | データ上1日に複数回存在する | 同じ日の起床時就寝前があるが今回は起床時のみ（新しいデータを更新しない） |
| Nutrition     | データ上1日に1回のみ        | その他のデータが異なる場合更新する                                       |
